# Architecture Icon Assets

This folder contains a curated subset of official cloud icon assets used by the in-browser architecture diagram generator.

Sources:
- AWS Architecture Icons: https://aws.amazon.com/architecture/icons/
- Azure Architecture Icons: https://learn.microsoft.com/azure/architecture/icons/
- Google Cloud Icons: https://cloud.google.com/icons

Notes:
- Assets were copied from the vendors' published icon packages and renamed for deterministic lookup.
- Keep these files only for architecture diagram illustration inside this product.
- If you update versions, replace files in `aws/`, `azure/`, and `gcp/` with equivalent official icons and keep stable file names.
